# Protótipo — Health Integration Core

Este repositório contém um protótipo navegável em HTML da plataforma **Health Integration Core** para demonstrações funcionais.

## Arquivos

- **Abrir protótipo HTML:** `prototipo-health-integration-core.html`
- **README do protótipo:** `README-prototipo.md`

## O que já vem no protótipo

1. Dashboard operacional com KPIs e incidentes
2. Catálogo de integrações
3. Flow Designer
4. Monitoramento com replay demo
5. Rules Engine
6. Visão de arquitetura

## Como executar

Basta abrir o arquivo HTML em qualquer navegador moderno:

```bash
xdg-open prototipo-health-integration-core.html
```

Ou, se preferir servir localmente:

```bash
python3 -m http.server 8080
# depois acesse: http://localhost:8080/prototipo-health-integration-core.html
```
